# 🚀 AI Career Assistant (Endee + AI Project)
“Endee vector database was used conceptually to store and retrieve embeddings. Due to project constraints, embeddings are handled in-memory using SentenceTransformers, simulating vector DB behavior.”
## 📌 Overview
This project is an AI-powered career assistant that:
- Matches resumes with job roles using semantic similarity
- Identifies skill gaps
- Suggests learning paths

## 🔥 Features
- AI-based job matching
- Skill gap analysis
- Resume PDF upload
- Career roadmap generator

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Endee (Vector DB concept)

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
